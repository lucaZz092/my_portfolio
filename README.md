## 🧪 Como visualizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/lucaZz092/portfolio.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd portfolio
   ```

3. Abra o arquivo `index.html` em seu navegador preferido.
